Inferences on the Encoding works of 3D Reconstruction

Based on THE SURVEY PAPER :  Image-based 3D Object Reconstruction: State-of-the-Art and Trends in the Deep Learning Era BY Xian-Feng Han*, Hamid Laga*, Mohammed Bennamoun Senior Member, IEEE

 

Models/Papers mentioned in the paper ( in order of precedence) :

 

1.     3D shapenets: A deep representation for volumetric shapes.

2.     Perspective Transformer Nets: Learning single-view 3D object reconstruction without 3D supervision.

3.     Learning a probabilistic latent space of object shapes via 3D generative- adversarial modeling.

4.     Rethinking re-projection: Closing the loop for pose-aware shape reconstruction from a single image.

5.     Deep disentangled representations for volumetric reconstruction

 

Basic Architecture

 

1.     Videos provide a temporal correlation between certian images.

2.     Additional data is in the silhouettes, segmentation masks, and semantic labels can be used to extract additional information.

3.     A look into output representaton :

1.     Volumetric Representation : Voxel grids are used. 2D convolutions can be extended to form such an output.

2.     Surface-based Representation : Meshes and Point Clouds. While memory efficient, hard to fit in current deep learning architectures.

4.     The backbone architecture is composed of an encoder and a decoder.

1.     The encoder maps the input into a latent variable x, referred to as a feature vector or code using sequence of convolutions.

2.     The decode aka the generator, decodes this via a number of upsampling operations.

3.     Fully connected layers in decoder are used for unstructed output while a deconvolution network is used for volumetric grids.

 

Encoding Stage

 

1.     A good mapping function, known as h should satisfy the following properties :

1.     Latent spaces of two inputs should be very close.

2.     A little change in the latent space should correspond to a small change in the actual input.

3.     Latent spaces must not incorporate extrinsic features such as the camera pose.

4.     A 3D model and its corresponding 2D images should be mapped onto the same point in the latent space. This will ensure that the representation is not ambiguous and thus facilitate the reconstruction.

2.      The first two conditions have been addressed by using encoders that map the input onto discrete or continuous latent spaces. These can be flat or hierarchical. The third one has been addressed by using disentangled representations. The latter has been addressed by using TL-architectures during the training phase.

 

Discrete Latent Spaces

Wu et al. in their seminal work introduced 3D ShapeNet, an encoding network which maps a 3D shape, represented as a discretized volumetric grid of size 303, into a latent representation of size 4000×1. Its core network is composed of nconv = 3 convolutional layers (each one using 3D convolution filters), followed by nfc = 3 fully connected layers. This standard vanilla architecture has been used for 3D shape classification and retrieval, and for 3D recon- struction from depth maps represented as voxel grids. It has also been used in the 3D encoding branch of the TL architectures during the training of 3D reconstruction networks.Both 3D shape and 2D image encoding networks can be implemented using deep residual networks (ResNet), which add residual connections between the convolutional layers, see for example. Compared to conven- tional networks such as VGGNet, ResNets improve and speed up the learning process for very deep networks.

 

Continuous Latent Spaces

Variational Autoencoders (VAE) and their 3D extension (3D-VAE) have one fundamentally unique property that makes them suitable for generative modeling: their latent spaces are, by design, continuous, allowing easy sampling and interpolation. The key idea is that instead of mapping the input into a feature vector, it is mapped into a mean vector μ and a vector of standard deviations σ of a multivariate Gaussian distribution. A sampling layer then takes these two vectors, and generates, by random sampling from the Gaussian distribution, a feature vector x, which will serve as input to the subsequent decoding stages.

 

Hierarchical latent spaces

To improve the quality of the reconstruction, Liu et al introduced a more complex internal variable structure, with the specific goal of encouraging the learning of a hierarchical arrange- ment of latent feature detectors. The approach starts with a global latent variable layer that is hardwired to a set of local latent variable layers, each tasked with representing one level of feature abstraction. The skip-connections tie together the latent codes in a top-down directed fashion: local codes closer to the input will tend to represent lower- level features while local codes farther away from the in- put will tend towards representing higher-level features. Finally, the local latent codes are concatenated to a flattened structure when fed into the task-specific models such as 3D reconstruction.

 

Disentangled representation

3D reconstruction can also benefit from dis- entangled representations where shape, pose, and lighting are represented with different codes. To this end, Grant et al proposed an encoder, which maps an RGB image into a shape code and a transformation code. The former is decoded into a 3D shape. The latter, which encodes lighting conditions and pose, is decoded into

(1) 80×80 RGB image with correct lighting, using upconvolutional layers

(2) camera pose using fully-connected layers (FC).

To enable a disentangled representation, the network is trained in such a way that in the forward pass, the image decoder receives input from the shape code and the transformation code. In the backward pass, the signal from the image decoder to the shape code is suppressed to force it to only represent shape. Zhu et al followed the same idea by decoupling the 6DOF pose parameters and shape. The network reconstructs from the 2D input the 3D shape but in a canonical pose. At the same time, a pose regressor estimates the 6DOF pose parameters, which are then applied to the reconstructed canonical shape. Decoupling pose and shape reduces the number of free parameters in the network, which results in improved efficiency.
